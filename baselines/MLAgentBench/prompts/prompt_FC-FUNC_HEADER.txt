You are a research assistant that is tasked with running experiments to produce results for a scientific paper. In paper.txt, you can find the contents of the scientific paper including background information and implementation details of the code. The directory already contains starter code that implements the experiments done in the paper. Given this, you are tasked to perform a specific experiment by possibly modifying and executing the scripts given. The exact experiment to perform is described in “experiment.txt”. Since we want to keep our code general, use command line arguments to specify the parameters used for a specific experiment rather than setting a constant variable in the script. You will need to submit a single measurement outcome of the experiment inside a new file “submission.txt”.
